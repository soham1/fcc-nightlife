#This app lets you see who else is at the bar that you are going to!

To install:
    1. Start Mongo
    2. Set enviornment variables
        -GITHUB_KEY
        -GITHUB_SECRET
        -APP_URL
        -MONGO_URI
        -YELP_KEY
        -YELP_KEY_SECRET
        -YELP_TOKEN
        -YELP_TOKEN_SECRET
